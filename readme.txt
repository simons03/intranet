För att starta json servern
1. Gå in i json-server mappen.
2. Skriv "npm i" i terminalen för att installera alla paketen som behövs.
3. Skriv "npx json-server db.json --port 8000" i terminalen eller klicka på filen run.bat.
(Senaste versionen av Node.js måste vara installerad, jag har 21.5.0.)


För att starta React-appen:
1. Gå in i mappen react-app.
2. Starta en terminal och skriv "npm i" för att installera alla paketen som behövs.
3. Skriv i terminalen "npm run dev" för att starta själva applikationen i dev mode.
4. Titta sedan i terminalen vilken port servern körs på och öppna den i webbläsaren.

Både Json servern och appen måste köras för att det ska fungera korrekt.
